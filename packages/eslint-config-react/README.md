## 使用方式

```bash
yarn add @catpawx/eslint-config-react --dev
```

```js
// .eslintrc.js
module.exports = {
  extends: ['@catpawx/eslint-config-react'],
  rules: {
    // 自定义你的规则
  },
}
```

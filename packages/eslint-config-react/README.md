## 使用方式

```bash
pnpm add -D @catpawx/eslint-config-react
```

```js
// .eslintrc.js
module.exports = {
  extends: ['@catpawx/eslint-config-react'],
  rules: {
    // 自定义你的规则
  },
}
```

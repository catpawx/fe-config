## 使用方式

```bash
yarn add @catpawx/eslint-config-preset --dev
```

```js
// .eslintrc.js
module.exports = {
  extends: ['@catpawx/eslint-config-preset'],
  rules: {
    // 自定义你的规则
  },
}
```
